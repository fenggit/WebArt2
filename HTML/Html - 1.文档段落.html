HTML（超文本标记语言——HyperText Markup Language）是构成 Web 世界的一砖一瓦。 它定义了网页内容的含义和结构。 除 HTML 以外的其它技术则通常用来描述一个网页的表现与展示效果（如 CSS），或功能与行为（如 JavaScript）。
